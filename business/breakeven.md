# breakeven.md

## Unit Economics & Break-even Analysis for Code-Sage

### Cost per Active User
- **Compute**: $5/user/month
- **Storage**: $2/user/month
- **Bandwidth**: $1/user/month
- **Total Cost per Active User**: **$8/user/month**

### Pricing Tiers
| Tier Name         | Price ($/mo) | Features                                                                 |
|-------------------|--------------|--------------------------------------------------------------------------|
| Basic             | $15          | Basic code analysis, limited recommendations, community support         |
| Professional      | $30          | Advanced code analysis, personalized recommendations, priority support   |
| Enterprise        | $50          | Comprehensive code analysis, custom recommendations, dedicated support   |

### Customer Acquisition Cost (CAC) Range
- **CAC Estimate**: $50 - $100 per user

### Lifetime Value (LTV) Estimate
- **Average Monthly Revenue per User**: $30 (assuming Professional tier as the most common)
- **Average Customer Lifespan**: 24 months
- **LTV Calculation**: 
  - LTV = Average Monthly Revenue per User × Average Customer Lifespan 
  - LTV = $30 × 24 = **$720**

### Break-even Users Count
- **Monthly Fixed Costs**: $2,000 (assumed for operations, marketing, etc.)
- **Break-even Formula**: Break-even Users = Monthly Fixed Costs / (Price per User - Cost per User)
- Using the Professional tier ($30):
  - Break-even Users = $2,000 / ($30 - $8) = $2,000 / $22 = **91 users**

### Path to $10K MRR
- **Target MRR**: $10,000
- **Assuming all users are on the Professional tier ($30/month)**:
  - Required Users = $10,000 / $30 = **334 users**
  
- **If targeting a mix of tiers**:
  - **Basic Tier**: 100 users = $1,500 MRR
  - **Professional Tier**: 150 users = $4,500 MRR
  - **Enterprise Tier**: 50 users = $2,500 MRR
  - **Total MRR**: $1,500 + $4,500 + $2,500 = **$8,500 MRR**
  
- **To reach $10K MRR**: 
  - Increase Professional tier users to 167 (total users = 417)
  - Or add more Enterprise tier users to increase MRR accordingly.

### Summary
- **Total Cost per Active User**: $8
- **Break-even Users Count**: 91 users
- **Path to $10K MRR**: 334 users on Professional tier or a mix of tiers to achieve the target.
